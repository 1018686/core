This project provides a set of components that may be used to add interactive maps to a Wicket application. These components wrap the OpenLayers 3 Javascript library and provide the necessary glue to let you manage them entirely in Wicket with Java code. More information may be found on the project's README page.

  • Simple - A map that displays the entire world
  • OpenStreetMap - A map that displays the entire world and used a tile layer from OpenStreetMap
  • Marker - A map that displays a marker over Miles' office
  • Popover - A map that displays a marker with a popover
  • WFS - A map that displays some WFS (Web Feature Service) data fetched via GeoJSON
  • Cluster - A map that displays some clustered data
